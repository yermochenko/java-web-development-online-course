# Web-разработка на Java

Курс рассчитан на студентов/слушателей, желающих углубить свои знания по языку программирования Java и научиться разрабатывать web-приложения с его использованием.

## Примерная программа курса

1. **Основы языка программирования Java.**

Понятие JVM: байт-код, структура памяти (Stack, Heap), сборщик мусора.

Типы данных: примитивные и ссылочные типы. Переменные и ключевое слово `var`. Операторы. Выражения. Управляющие конструкции (ветвления и циклы).

Классы-оболочки. Массивы.

2. **Объектно-ориентированное программирование. Часть 1.**

Классы. Объекты . Пакеты.

Поля. Финальные поля, параметры, локальные переменные. Методы: передача параметров в методы, возврат значений из методов, методы с переменным числом параметров.

Ключевое слово `this`. Статические поля и методы. Блоки инициализации и статические блоки инициализации.

Конструкторы и использование ключевого слова `this` в конструкторах.

Инкапсуляция. Области видимостри и модификаторы доступа. Get-теры и set-теры. Immutable-классы. JavaBeans.

Наследование. Наследование vs композиция. Приведение типов в иерархии наследования. Класс `Object` как вершина иерархии классов в Java. Оператор `instanceof` и метод `Object.getClass()`. Конструкторы при наследовании. Ключевое слово `super` в методах и конструкторах.

Полиморфизм. Перегруженные и переопределённые методы. Финальные методы и классы. Перегрузка методов `Object.toString()`, `Object.equals()`, `Object.hashCode()`, `Object.clone()`.

3. **Объектно-ориентированное программирование. Часть 2.**

Абстрактные классы. Интерфейсы.

Анонимные классы. Внутренние и вложенные классы. Функциональные интерфейсы. Лямбда выражения. Ссылки на методы.

Перечисления. 

4. **Принципы SOLID.**

Single Responsibility Principle - Принцип единственной ответственности.

Open-Closed Principle - Принцип открытости/закрытости.

Liskov Substitution Principle - Принцип подстановки Лисков.

Interface Segregation Principle - Принцип разделения интерфейса.

Dependency Inversion Principle - Принцип инверсии зависимостей.

Многослойная архитектура. Инверсия зависимостей и внедрение зависимостей. Шаблоны проектирования Factory Method, Abstract Factory, Builder, ServiceLocator. Понятие IoC-контейнера.

5. **Обработка исключений.**

Иерархия исключений. Исключения vs ошибки. Проверяемые и непроверяемые исключения. Обработка исключений. Блок `try`-`catch`-`finally`. Объявление генерируемых методом исключений с помощью ключевого слова `throws`. Генерация исключительной ситуации с помощью ключевого слова `throw`. Собственные исключения. Пробрасывание и маскирование исключений.

6. **Maven.**

7. **Модульное тестирование.**

Библиотека JUnit. Тестовые методы и аннотация `@Test`. Утверждения и класс `Assertions`. Тестирование возникновения исключений. Изолирование тестов. Test Driven Development. Пирамида тестирования.

8. **Журналирование.**

Библиотека Log4j. Основные концепции. Конфигурирование. Уровни логгирования. Класс `Appender` и `Layout`.

9. **Обработка строк.**

Класс `String`. Особенности хранения. Строковые литералы.

Классы `StringBuilder` и `StringBuffer`.

Регулярные выражения. Классы `Pattern` и `Matcher`.

Локализация и интернационализация. Парсинг и форматирование чисел, дат и времени.

Обработка дат и времени.

10. **Обобщения (Generics).**

Обобщённые классы и обобщённые методы. Ограничение на обобщённые типы. Символ `?` при описании обобщённых методов.

11. **Коллекции.**

Интерфейсы `Iterator` и `Iterable`. Итерация объектов с помощью цикла `for`.

Интерфейсы `Collection`, `List`, `Queue` и `Deque`. Классы `ArrayList`, `LinkedList` и `ArrayDeque`.

Интерфейсы `Comparable` и `Comparator` и алгоритмы класса `Collections`.

Интерфейсы `Set` и `Map`. Классы `HashSet` и `HashMap`. Особенности переопределения методов `equals()`, `hashCode()`, `compare()` и `compareTo()`.

12. **API потоковой обработки данных.**

Потоковое API. Функционльные интерфейсы, используемые в потоковом API. Класс `Optional`.

13. **Потоковый ввод/вывод.**

Иерархия потоковых классов. Байтовые и символьные потоки. Чтение файлов. Строковые потоки. Буферизация. Конвертация потоков.

Блок try-with-resources.

Сериализация.

Форматированный ввод и вывод.

Класс `File`.

14. **Обработка данных в формате JSON.**

Формат JSON. Библиотека Jackson. Преобразование Java-объектов в JSON. Преобразование JSON в Java-объекты, в том числе в коллекции объектов.

15. **Реляционные базы данных и SQL**

СУБД PostgreSQL. Язык SQL: DDL, DML.

Понятие транзакции. Принципы ACID. TCL.

16. **JDBC**

Понятие драйвера. Установка соединения с СУБД. Интерфейс `Connection`.

Запросы. Интерфейсы `Statement` и `PreparedStatement`. Выполнение запросов.

Обработка результатов запроса. Интерфейс `ResultSet`.

Соответствие типов данных Java и типов данных SQL.

Работа с транзакциями в JDBC.

Пул соединений.

Шаблон проектирования Data Access Object. Переменные потока и класс `ThreadLocal`.

17. **Servlet API.**

Понятие сервлета и его жизненный цикл.

Контейнер сервлетов Tomcat, настройка и запуск.

Развёртывание сервлетов в контейнере сервлетов. Дескриптор развёртывания. Аннотация `@WebServlet`.

Методы HTTP и их обработка в сервлетах. Интерфейсы `HttpServletRequest` и `HttpServletResponse`.

Понятие HTTP статуса. Методы `setStatus()`, `sendRedirect()` и `sendError()`.

Обработка данных HTML-форм.

Диспетчерезация запросов: `include()` и `forward()`. Передача данных между сервлетами с помощью атрибутов.

Шаблон проектирования Model-View-Controller.

18. **Java Server Pages (JSP).**

Понятие JSP и её жизненный цикл. JSP-теги. Expression Language (EL) и библиотека JSP Standard Tag Library (JSTL). Работа с атрибутами запроса.

Создание пользовательских тегов.

Шаблон проектирования Command.

19. **HTML. CSS.**

20. **Сессии. Фильтры. События.**

Понятие сессии и её жизненный цикл. Настройка сессий. Аутентификация на основе сессий. Cookies.

Фильтры. Фильтр для перекодирования запроса. Фильтр для авторизации.

События и обработчики событий. Инициализация приложения. События входа в систему и выхода из системы.

21. **REST.**

Понятие REST. Применение методов HTTP при проектировании REST API. Использование HTTP статусов в REST. Тестирование REST API в Postman.

22. **JavaScript.**

Базовые понятия JavaScript. Работа с DOM.

Синхронные и асинхронные методы. Понятие `Promise`. Ключевые слова `async` и `await`.

Fetch API и работа с REST API.

23. **Spring Framework.**

Основные понятия Spring Framework: Spring MVC, Spring Data JPA.
